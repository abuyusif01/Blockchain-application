# Dependencies

You must have nodejs and npm installed your system the follow the above
instructions.

- `isLinux ? sudo npm install -g truffle : npm install -g truffle`
- `npm install dotenv`
- `npm install env`
- Also get ganache from here[(https://trufflesuite.com/ganache/)]
